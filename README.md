# nmlocal.co.uk

The website for **NM Local** — a free monthly community magazine hand-delivered
to homes across Newton Mearns, East Renfrewshire.

Deployed automatically to Porkbun Static Hosting from the `main` branch.

## Pages
- `index.html` — home
- `advertise.html` — founding advertiser rates (FAQ schema)
- `directory.html` — local business directory (launches with Issue 1)
- `articles/` — three schema-marked articles for readers
- `media-kit.pdf` — the one-pager linked from sales emails

## Editing
Plain HTML and one stylesheet (`styles.css`) — no build step. Edit, commit,
push, and Porkbun redeploys.
